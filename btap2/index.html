What is CSS?

Cascading Style Sheet (CSS) is used to format the layout of a pageweb.

With CSS, you can control the color, font, the size of tex, the spacing beetween elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different device and screen sizes, and much more!

Tip: The word cascading means that a style applied to a parent elements will also apply to all children elements within the parent. So, if you set the color of the body text to "blue" , all headings, paragraphs, and other text elements within the body will also get the same color(unless you specify somwthing else)!
